# Placement Analytics Project

A complete data analytics project that analyzes factors influencing engineering student placements.

## Dataset
Source: Kaggle Engineering Placement Dataset

## Tools Used
- Python (Pandas, Matplotlib, Seaborn, Scikit-learn)
- Power BI
- Excel

## Problem Statement
To analyze what factors influence placement success and build predictive and visualization models.

## Analytics Work
- Data cleaning & feature engineering
- Placement probability analysis
- Internship, CGPA, Backlog & Stream impact analysis
- Built interactive Power BI dashboards

## Key Insights
- Students with internships have significantly higher placement rates
- High CGPA (>7) improves placement probability
- Backlogs reduce placement chances
- CS & IT streams have the highest placement rates

## Business Takeaway
Institutions should emphasize internships, certification learning, and academic performance to improve placement success.
